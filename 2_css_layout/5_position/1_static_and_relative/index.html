Introduction CSS Position

  • static adalah nilai default dari tiap-tiap elemen ketika tidak diberi properti position (nilai default)
  • menggunakan position selain static, akan membuat sebuah elemen menjadi seolah-olah berbeda dimensi dari elemen lainnya
  • elemen yang diberi position selain static dapat menggunakan properti top, left, bottom right untuk mengatur posisinya

Terdapat empat jenis position

  • static - nilai default
  • relative -
  • absolute
  • fixed

TOPIC 1 : POSITION RELATIVE

  • ketika menggerakkan elemen dengan posisi relative(menggunakan properti top, left, bottom dan right), ruang yang ditempati oleh elemen tersebut masih ada
  • ketika kita menggerakkan elemen dengan posisi realtive, elemen akan bergerak relatif terhadap posisi semulanya
  • jiak kita memberi properti top: 0; dan left: 0; maka elemen tidak berubah posisinya
