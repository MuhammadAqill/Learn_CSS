Hal yang anda perlu tahu linear-gradient

  • bukan property css3
  • melainkan value untuk background-image
  • background-image: linear-gradient( {warna awal}, {warna akhir} );





contoh penggunaan `linear-gradient`






multiple warna

  • background-image: linear-gradient( {warna awal}, {warna-2}, ..., {warna akhir} );






Mengguakan sintaks lengkap

  • background: linear-gradient( {sudut} atau {arah}, {warna awal}, {warna akhir} )
