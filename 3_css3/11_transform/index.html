Properti Transform Digunakan Untuk Memanipulasi Format Visual Dari Elemen HTML

cara penulisan

  • transform: {fungsi}
  • fungsi:-
    • scale - memperbesarkan / memperkecilkan
      • scale(angka)
      • scaleX(angka)
      • scaleY(angka)
    • rotate
      • rotate(deg)
      • rotateX(deg)
      • rotateY(deg)
    • skew - membuat condong / miring
      • skew(deg)
      • skewX(deg)
      • skewY(deg)
    • translate - mengubah posisi
      • translate(translate)
      • translateX(translate)
      • translateY(translate)
