Hal yang anda perlu tahu radial-gradient

  • bukan property css3
  • melainkan value untuk background-image
  • background-image: radial-gradient( {warna awal}, {warna akhir} );





contoh penggunaan `radial-gradient`






multiple warna

  • background-image: radial-gradient( {warna awal}, {warna-2}, ..., {warna akhir} );






Penggunaa color stop






Mengguakan sintaks lengkap

  • background: radial-gradient( {tipe} {jangkaun} at {posisi-y} {posisi-x}, {warna 1}, {warna 2} ) ...{warna-n}
  • tipe
    • ellipse(default)
    • circle
  • jangkaun
    • jarak dari pusat gradient ke sisi terluar-nya
      • closest-corner - jarak dari pusat gradient ke susut terdekat elemen
      • closest-side - jarak dari pusat gradient ke sisiterdekat elemen
      • farthest-corner - jarak dari pusat gradient ke sudut terjauh elemen
      • farthest-side - jarak dari pusat gradient ke sisi terjauh elemen
