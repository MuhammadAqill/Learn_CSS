Menggunakan Poperti optional

box-shadow: {inset} {x-offset} {y-offset} {blur} {spread} {color}

  • inset - membuat bayangan berada di dalam kotak
  • x-offset - mengatur jarak bayangan secara vertikal
  • y-offset - mengatur jarak bayangan secara horizontal
  • spread - ukuran dari bayangannya

  • menggunakan spread bayangannya akan bertambah besar 5px
  • box-shadow: 5px 5px 5px 5px rgba(0, 0, 0, 0.5);
